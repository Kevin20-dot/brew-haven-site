 Overview
A modern coffee shop website designed to showcase Brew Haven Coffee.  
The site features a cozy, welcoming design with earthy tones and includes pages such as Home, About Us, Menu, Gallery, and Contact.  
It highlights the shopâ€™s mission to serve fresh brews, showcases a variety of coffee and snacks, and provides easy navigation for visitors.  
Built using no-code/AI-powered web design tools, the website is mobile-friendly and SEO-ready.

## Features
- Hero section with tagline and call-to-action
- About Us page with shop story and mission
- Menu showcasing coffee and snacks
- Gallery with shop and product images
- Contact page with form and embedded map
- Footer with social media links and opening hours
